## Estrutura

Importe as partes e componha-as.

```tsx
import * as CheckboxGroup from "@diceui/checkbox-group";

<CheckboxGroup.Root>
  <CheckboxGroup.Label />
  <CheckboxGroup.List>
    <CheckboxGroup.Item>
      <CheckboxGroup.Indicator />
    </CheckboxGroup.Item>
  </CheckboxGroup.List>
  <CheckboxGroup.Description>
</CheckboxGroup.Root>
```

## Checkbox animado

<Steps>
  <Step>
    Atualize o `tailwind.config.ts` para incluir a seguinte animação:

    ```ts
    export default {
      theme: {
        extend: {
          keyframes: {
            "stroke-dashoffset": {
              "0%": { strokeDashoffset: "100%" },
              "100%": { strokeDashoffset: "0" },
            },
          },
          animation: {
            "stroke-dashoffset": "stroke-dashoffset 0.2s linear forwards",
          },
        },
      },
    }
    ```
  </Step>
  <Step>
    Copie e cole o bloco `CheckboxGroup.Indicator` do exemplo abaixo no seu projeto.

    <ComponentTabs name="checkbox-group-animated-demo" />
  </Step>
</Steps>
