## Principais recursos

- **Componíveis por design** — Crie interfaces complexas combinando componentes simples e focados
- **Construído sobre o shadcn/ui** — Usa os mesmos princípios de design e abordagem de estilos do shadcn/ui
- **Prontos para copiar e colar** — Mesma experiência de instalação dos componentes do shadcn/ui
- **Totalmente personalizáveis** — Construídos com Tailwind CSS e pensados para se adequar ao seu design system
- **Funcionalidades avançadas** — Componentes complexos como tabelas de dados, players de mídia e interfaces de arrastar e soltar

## Por que o Dice UI?

O Dice UI é uma coleção de componentes totalmente componíveis que estendem o shadcn/ui. Esses componentes são acessíveis e personalizáveis para atender às suas necessidades. O Dice UI está pronto para copiar e colar, sem necessidade de configurações complexas.
Cada componente segue as diretrizes da WCAG, inclui os atributos ARIA corretos e oferece navegação por teclado pronta para uso.
